We are going to develope functions to help us display each asteroid's name and its distance from the earth, in a table, on a webpage. The first thing that might help us to do this, is to have direct access to an array that contains the information for each asteroid. The function below returns the array that we want. This array prevents us from having to retype response.near_earth_objects[date][i] each time we want to access one of the asteroid objects, by creating a new array, of which each index has a single asteroid object.

Now that we have some functions to return the info we want, let's write a function that iterates through the array, calls these functions, and uses the returned data to create a table on our webpage.

The printAsteroidDetails function starts out by setting up our table elements. The function then iterates through the array, each time calling the name and distance functions and adding the returned information to the corresponding cells for that row. The end result is a complete table with 2 columns and a number rows equal to the number of asteroids in our array. You can view an example of my page that implements these functions, HERE.
